## EcoNuker

EcoNuker is a Guilded economy bot! We focus solely on the economy and fun side of things, meaning no real moderation commands!

Some things we make:
- Economy
- Minigames
- Fun commands
- Memey commands
- APIs
    - Libraries that work with our APIs!!!
